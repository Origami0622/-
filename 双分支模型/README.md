​        该项目为我硕士期间的研究课题，相比单乐器识别，复调场景中多个乐器同时发声，频谱高度重叠，传统模型很难准确区分主导乐器，因此识别难度明显更高。
在这个项目中，我主要负责：

1. 模型架构设计
2. 特征建模方案优化
3. 多模型决策融合策略设计
4. 实验验证与性能分析

​        整个系统的核心思路是 “双模型互补 + 决策级融合”。第一条分支使用 CNN 模型，主要基于 MFCC 特征进行建模，擅长捕捉局部时频结构信息；第二条分支使用 LB-Wav-KAN 模型，它是在 KAN 网络基础上引入小波函数构建的结构，更偏向于增强非线性表达能力和时频多尺度建模能力。
​        基于项目需求，我开发构建了从特征提取、模型构建与实现，模型训练到效果验证一条完整的链条，并在最后将其封装为可执行的本地程序文件，程序的效果为识别用户输入进去的音频数据，并对其中可能包含的乐器类别进行预测，最后基于用户预测结果。该项目成果最终发表为EI会议论文（第一作者）。
